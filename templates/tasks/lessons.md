# Lessons

Use this file when the assistant learns from a correction.

The best lessons are short behavior rules.

Lessons live under `tasks/` because they change how the assistant does future work.

## Lesson Template

## Examples

- When I ask for a quick answer, answer first and explain only if needed.
- Before changing public-facing copy, check the audience and desired tone.
- If a task touches privacy, run the privacy checklist before calling it done.
