# Concepts

Living synthesis, one file per idea you actually use. A concept is your current best understanding, not a copy of any single source.

Update concepts as your thinking changes. The confidence label and update notes show how solid each one is and how it got there.

- Template: `../concept-template.md`
- Built from: briefs in `../briefs/` and sources in `../sources/`.
- Track changes over time in `../research-log.md`.
