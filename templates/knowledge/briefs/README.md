# Briefs

Short summaries, one per source or topic: the core idea, the useful points, what it adds, and what is still open.

A brief is the bridge between a raw source and a living concept. It is where you decide what actually matters before it shapes your thinking.

- Template: `../brief-template.md`
- Pulls from: files in `../sources/`.
- Feeds: concepts in `../concepts/`.
