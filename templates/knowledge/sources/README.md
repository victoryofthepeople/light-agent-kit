# Sources

Raw material, one file per source: an article, video, talk, course note, or conversation.

Capture what the source actually says here. Keep quotes short and attributed. This is the layer closest to the original, so it is where provenance lives.

- Template: `../source-template.md`
- Next step from a source: write a brief in `../briefs/`, or add to a concept in `../concepts/`.
- Privacy: source notes from private calls, paid courses, or client material stay in your private workspace. See `PRIVACY-CHECKLIST.md`.
