# Light Agent Kit Workspace

This is the private workspace for the assistant.

Start with `home.md`. It shows what matters now, what the assistant knows, what it can do, what needs approval, and what power may come next.

## Read Order

When starting a session, read:

1. `home.md`
2. `profile/me.md`
3. `profile/goals.md`
4. `profile/preferences.md`
5. `profile/taste-and-standards.md`
6. `profile/capability-map.md`
7. `profile/power-map.md`
8. `profile/permission-map.md`
9. `tasks/active.md`
10. `tasks/lessons.md`

Then read relevant `projects/`, `knowledge/`, `daily/`, `ideas/`, or `power/` files as needed.

## What Lives Here

- `home.md`: current front page.
- `profile/`: durable context, preferences, standards, capabilities, power, and permissions.
- `tasks/`: active work, completed work, and lessons.
- `daily/`: dated notes and breadcrumbs when useful.
- `projects/`: deeper context for work, business, clients, or ideas.
- `ideas/`: raw capture.
- `knowledge/`: reusable notes, research, and source-backed concepts.
- `power/`: activation cards for new powers before they go live.
- `skills/`: repeatable workflows if copied into this workspace.

## Operating Rules

- Keep this workspace private.
- Keep public examples fictional.
- Ask before sending, publishing, deleting, spending, changing calendars, using secrets, or connecting new apps.
- Save corrections as short lessons in `tasks/lessons.md`.
- Refresh `home.md` after meaningful changes.
