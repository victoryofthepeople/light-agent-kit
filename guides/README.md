# Guides

These guides are optional.

Start by proving value in the workspace. Add tools when they solve a real problem. For a plain rundown of what Light Agent Kit can help an agent do, read [../CAPABILITIES.md](../CAPABILITIES.md).

## Recommended Order

1. Use [../START-HERE.md](../START-HERE.md) or [../ONBOARDING.md](../ONBOARDING.md).
2. Fill in the templates.
3. Try the daily loop for a few days.
4. Add a notes app, phone access, or app connections only when useful.
