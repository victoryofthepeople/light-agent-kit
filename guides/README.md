# Guides

Optional setup for when the starter workspace is working and you want more.

Read [../EXPAND.md](../EXPAND.md) first if you are an AI agent helping someone grow their workspace.

## Recommended order

1. Finish the guided setup in [../site/](../site/) and use the workspace for a few days.
2. Give your AI this repo when you want to add structure, skills, or integrations.
3. Read [first-week.md](first-week.md) before adding tools.
4. Add phone access, app connections, or a stronger runtime only when the value is clear.
