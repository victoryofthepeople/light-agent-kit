# Using Light Agent Kit With Obsidian

Obsidian is useful if you want your kit workspace to feel like a private notes vault.

## Simple Setup

1. Create a new Obsidian vault.
2. Copy the Light Agent Kit folder structure into the vault.
3. Fill in `profile/me.md`, `profile/goals.md`, and `profile/preferences.md`.
4. Use daily notes for the daily loop.
5. Use links between projects, knowledge notes, and tasks when helpful.

## Suggested Vault Shape

```text
profile/
tasks/
daily/
projects/
ideas/
knowledge/
skills/
```

## Good First Habit

Create one daily note each morning:

```text
daily/YYYY-MM-DD.md
```

Then ask your AI assistant to help you plan from that note.

## Safety Notes

- Keep the vault private by default.
- Use a separate public repo for templates or examples.
- Review sync settings before storing private data.
