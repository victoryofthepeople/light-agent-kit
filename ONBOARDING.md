# Onboarding

**This flow moved.**

The main onboarding is the guided setup site in [`site/`](site/). It asks questions in the browser and builds your starter workspace folder as a download. No paste-the-repo step.

## Start here instead

1. **Humans:** run the site locally (`cd site && npm install && npm run dev`) or use the deployed version when available.
2. **After you have a workspace folder:** give your AI this GitHub repo if you want help growing it. Point the agent at [EXPAND.md](EXPAND.md).

## Legacy agent-led onboarding

The old interview flow — paste the repo, say "onboard me," build the full kit in chat — is archived at [examples/legacy-agent-onboarding.md](examples/legacy-agent-onboarding.md). It is not the default path anymore.
