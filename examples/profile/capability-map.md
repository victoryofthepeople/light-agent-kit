# Capability Map

Fictional example.

What this assistant can do now. It pairs with `power-map.md` and `permission-map.md`.

## Current Mode

- [x] Files and chat only
- [x] Notes workspace
- [ ] Phone access
- [ ] Read-only app access
- [ ] Write access with approval
- [ ] Runtime automation

## Can Read

- Light Agent Kit workspace files.
- Daily notes the user adds.
- Project files the user shares.
- Local Markdown notes in the user's private notes folder.

## Can Suggest

- Daily plans.
- Project next steps.
- Task organization.
- Copy drafts.
- Research questions.
- Lessons drawn from corrections.

## Can Change With Approval

- `home.md`
- `tasks/active.md`
- `tasks/done.md`
- `tasks/lessons.md`
- `projects/*`
- `knowledge/*`
- `profile/*`
- `power/*`

## Can Change Quietly

- Add low-risk notes to `ideas/inbox.md`.
- Add a daily note when the user asks for a daily plan.
- Add a lesson after an explicit correction.
- Refresh `home.md` from approved workspace files.

## Not Connected Yet

- Gmail.
- Calendar.
- Telegram.
- Obsidian sync.
- Scheduled runtime automation.

Clear yes rules live in `permission-map.md`.
